# ipyrad2 Documentation

This site is the user guide and command reference for ipyrad2. It covers installation, files and data types, assembly workflows, output exports, task-focused recipes, and downstream analyses.

## Start Here

If you are new to the project, begin with [What Is ipyrad2?](./getting-started/what-is-ipyrad.md). It provides the project overview and directs you to the most useful next steps.

## Find What You Need

- [Installation](./getting-started/installation.md) — install ipyrad2 and verify your environment.
- [Files and Data Types](./getting-started/files-and-data-types.md) — understand supported inputs, outputs, and workflow entry points.
- [Assembly Quick Guide](./assembly/index.md) — follow the assembly workflow from input data to assembled outputs.
- [Writing Outputs](./writing-outputs/index.md) — filter and export data for external tools.
- [Recipes](./recipes/index.md) — find focused examples for specific tasks.
- [Analysis Guide](./analyses/index.md) — choose and run downstream analyses.
